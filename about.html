About Me

I’m Gary Samuel, a full-stack software developer and Computer Science student at California State University, Fullerton (3.6 GPA, graduating Dec 2026). I specialize in building performant, scalable, and maintainable systems across e-commerce, real-time applications, and developer tooling.

At Bimmer Plug, I delivered a full Shopify UI redesign with modular, reusable Liquid and JavaScript components—boosting load speed by 15% and reducing code duplication by 40%. I engineered dynamic, API-driven product pages for a 700+ SKU catalog, integrated them with the Shopify Storefront API, and automated product workflows via the Admin API, cutting manual processes and minimizing data errors.

I also developed AutoFix Bug Fixer, a Python CLI tool that integrates AST parsing, pytest failure analysis, and traceback parsing to detect type mismatches, unused debug prints, and other defects—outputting targeted fix suggestions without altering the original codebase.

My technical stack includes C, C++, Python, Java, JavaScript, PHP, Liquid, and Assembly, with frameworks like React.js, Vue.js, Angular, Django, Flask, FastAPI, and expertise in REST APIs, Shopify APIs, NumPy, Pandas, TensorFlow, PyTorch, and Socket.IO. I apply clean architecture principles, performance profiling, and test-driven development to ensure codebases remain extensible, debug-friendly, and production-ready.

Outside of professional work, I’ve taught AP Computer Science A and AP Calculus AB, mentored students on algorithm design, and conducted debugging workshops—strengthening my ability to communicate complex technical concepts effectively.
